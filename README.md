# super-egg

> A full stack node project about [egg](https://eggjs.org/zh-cn/intro/quickstart.html)

## 开发

```bash
npm install
npm run dev
open http://localhost:7001/
```

## 部署

```bash
// 打包
npm run build

// 启动服务
npm start
open http://localhost:7001/
```

## 介绍

本地前端工程化开发，集成 node 自动化部署，从开发到最终部署一气呵成
